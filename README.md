## How to Run ?

- Download the Project, Unzip and Copy to `C:xampp/htdocs/git/project_rakamin`
- Open XAMPP and Start **Apache** and **MySql**
- Import the file database to your own database with the name of the database is [api_simple_chat](https://github.com/Dradieco/project_rakamin/tree/main/database) 
- Open Postman or click [link](https://www.postman.com/rickyraven20/workspace/mini-project-rakamin) for the reference to use the API
